# About
Check if a username is available in Monkeytype or not.

# Data Management
We don't take any data from you, including your personal information such as legal name, house address, credit card numbers, etc... Monkeytype Namecheck is a website designed to help you check if a username is taken on Monkeytype in a fast way without any signing up, or logging in. It would be pointless to add tracking.

More information; [Privacy Policy page](https://www.mtgoals.cc/pages/policy)
